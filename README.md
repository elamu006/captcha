# 🔐 CAPTCHA Generator & Validator

A simple and responsive CAPTCHA Generator built using **HTML**, **CSS**, and **JavaScript**. This project generates random CAPTCHA codes and validates user input to help prevent automated bot submissions.

## 🚀 Live Demo

🌐 Live Website: https://elamu006.github.io/captcha/

## 📖 About the Project

This CAPTCHA application generates a random verification code that users must enter correctly before proceeding. It demonstrates basic front-end security concepts and JavaScript DOM manipulation.

CAPTCHA (Completely Automated Public Turing test to tell Computers and Humans Apart) is commonly used to distinguish human users from bots.

## ⚙️ How It Works

1. A random CAPTCHA string is generated using JavaScript.
2. The CAPTCHA is displayed on the screen.
3. The user enters the displayed CAPTCHA.
4. JavaScript compares the input with the generated CAPTCHA.
5. A success message appears if the CAPTCHA matches.
6. An error message appears if the CAPTCHA is incorrect.
7. Users can refresh the CAPTCHA to generate a new code.
